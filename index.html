Stage 0: Getting Started with HTML

The World Wide Web

The World Wide Web is a collection of HTML documents.

HTML stands for HyperText Markup Language. HTML documents are the basis for everything on the Web.

Components of the web: (1) a browser = a program that runs on your computer to make requests to servers via the Internet using a protocol called HTTP and to display files found on the web; (2) the Internet = the world's largest computer network; (3) servers = computers optimized for hosting files that make up the web.


HTML

HTML is made up of text content (plain text that you see), markup (how the text is arranged), references to other documents (e.g., images and videos), and links to other pages.


Markup

Markup is made up of tags.

A tag contains an opening tag and, unless it is a void tag, it also contains a closing tag.

The contents within a tag could include other tags.

An attribute is a property of an HTML element.

An HTML element is everything within a set of opening and closing tags. HTML elements can be either inline or block. Block elements create an invisible box around the contents, and you can edit the dimensions of that box.


Computers are "Stupid"

Coding entails writing text in a language that a computer can understand.

Computers are sensitive to typos. Small mistakes that a "smart" human could easily correct can create huge problems when given to a "stupid" computer. Thus, programmers must write exactly the way a computer understands (with correct syntax).
